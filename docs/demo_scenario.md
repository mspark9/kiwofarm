# 발표 시연 시나리오

**페르소나**: 옥천에 정착하려는 부부 "박경수" (45세, 자본 5천만원, 농지 1,500평 임대 예정)

**목표 시간**: 1분 30초

## 5단계 흐름

1. **모드 선택** — "귀농" 카드 클릭
2. **조건 입력** — 지역: 옥천 / 자본: 5천만 / 면적: 1,500평 / 시설: 비닐하우스 / 노동력: 2인
3. **작목 TOP 3** — 토마토 / 고구마 / 블루베리 + GPT-4o 추천 이유 말풍선
4. **디지털 트윈** — 토마토 클릭 → 12개월 매출/노동 차트 + 7월 흰가루병 위기 타임라인 + AI 코치
5. **출하 대시보드** — ★★★★ 오늘 vs ★★★★★ 3일 후 + 14일 가격 예측 + 옥천 우수농가 9/12 패턴
